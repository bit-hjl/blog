选择排序+冒泡排序+插入排序介绍

# 一、选择排序

选择排序是一种比较容易理解的排序算法，它的做法是每次从当前元素中选取最小的元素排在当前未排好序序列的开头，依次向后推实现排序。 特点：不稳定 时间复杂度：最好、最坏、平均时间复杂度均为O（n^2） 代码如下：

void sort(int num[])
{
	for(int i = 0;i<n;i++)
	{
		for(int j = i + 1;j <n;j++)
		{
			int k = -1;
			if(num[j] < num[i])
			k = j;
			if(k != -1)
			{
				num[i] ^= num[k];
				num[k] ^= num[i];
				num[i] ^= num[k];
			}
		}
	}
}

# 二、冒泡排序

冒泡排序的思想是遍历数组时将当前元素和下一个元素比较，若当前元素更大，则将该元素和下一个元素交换位置，不难看出，每遍历一次可以将未排好序的序列中最大的元素置于序列末尾。 代码实现：

void sort(int num[])
{
	for(int i = 0;i < n;i++)
	{
		for(int j = 1;j < n - i;j++)
		{
			if(num[j] < num[j-1])
			{
				num[j] ^= num[j-1];
				num[j-1] ^= num[j];
				num[j] ^= num[j-1];
			}
		}
	}
}

# 三、插入排序

顾名思义，插入排序即把当前元素插入到合适的位置，使其满足排序的条件。我们可以从第二个元素开始（此时认为第一个元素单独构成了一个已经排好序的序列），依次插入已排好序的序列中。由于当前元素x已经拿出，我们只需要在之前的序列中找出一个位置，使得它前边的元素比x小，后边的元素比x大即可 代码实现：

void sort(int num[])
{
	for(int i = 1;i<n;i++)
	{
		int swap = num[i],j;
		for(j = i-1;j >= 0;j--)
		{
			if(swap < num[j])
			num[j + 1] = num[j];
			else
			break;
		}
		num[j+1] = swap; 
	}
}
