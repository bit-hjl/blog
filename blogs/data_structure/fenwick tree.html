树状数组

vuePress-theme-reco bit-hjl    2021

树状数组

bit-hjl 2019-06-29 数据结构

介绍

以O（log(n)）时间复杂度进行区间求和及修改的数据结构

# 一、基本原理

树状数组是一个可以以高速查询前缀和的一种方法，对于正常的前缀和数组来讲，如果改变其中一个值，他的修改的复杂度为O（n），而查询的复杂度为O(1),对于数据庞大且需要频繁修改的一组数据可以用树状数组来实现，它的修改复杂度和查询复杂度都是O（log n），效率有较大的一个提升。树状数组不会提升代码的空间复杂度。

首先对于任意一个整数，他都具有唯一一个二进制的表达形式，而对于每一位，都有且仅有0，1两种形式。因此，我们可以通过这种形式对数组进行划分。对于10（1010），可以划为1（0001）到8（1000），9（1001）到10（1010），因此，对于树状数组tree，tree[i] = num[i]。其中，lowbit(x),即为x在二进制情况下从最后一位向前第一位为1所形成的数。如对于10，lowbit(10) = 2(10),对于任意一个奇数（二进制下最后一位恒为1）Lowbit（奇数） = 1，对于2^n,由于仅在最高位为1，所以lowbit(2^n) = 2^n,为它本身。

这样，我们就把数组区间上点的修改和求和转变为了tree数组的求和和修改。

# 二、lowerbit实现

lowerbit有两种实现方式：

1、x&(-x) : 对于-x,它的二进制编码是x的反码加一，即将x中的1换0，0换1，之后再加一。对于原来位上为0的位，-x对应位为1，加1变0，进位同时运算得0.当且仅当原位为1时，由于转换为0，但是之前有进位，使得-x在该位上也为1，得1.之后不再进位，所以对后序计算结果均为0，成功实现。

2、 x&(x^(x-1)):(x-1)可以使得x从最后一位向前首次为1及其之后的数与之前取反，由于x^(x-1),之前没有变化的数全部归零，变化的全部取1，再次&x时，可以实现该操作。

int lowbit(int i)//lowbit函数可以返回最低位为1的那个数，如8为1000，返回8，5为101，返回1。6为110，返回10（2），即2.
{
return i&(-i);//return i&(x^(x-1))
}

# 二、单点修改

该函数用于修改tree数组，以及建树操作。显然对于修改的第x个数据，在x到n的某些数据可能受到影响。第i个数首次在tree[i]中出现，之后的数组若与其有关，则显然在i + lowbit(i)*整数后的数组与num[i]有关

代码实现：

int change(int i,int k)//修改函数，即将第i个数据增加k以后用于更新tree数组和前缀和，也是用来建树的函数
{
	while(i <= n)
	{
		tree[i] += k;	
		i += lowbit(i);
	}
}

# 三、区间和查询

对于输入的i，tree[i]中存储了i - lowbit(i) + 1到i的值，因此要求序列和还需要加上1到i - lowbit(i)的值，以此类推，可以求得序列和代码实现：

int search(int i)//查询函数，可以用于查询前i个数的和
{
    int sum = 0;
    while(i)
    {
        sum += tree[i];
        i -= lowbit(i);
    }
    return sum;
}

# 四、整体测试

#include<stdio.h>
#include<string.h>
int n;
int num[100];
int tree[100];
int all[100];
int lowbit(int i)//lowbit函数可以返回最低位为1的那个数，如8为1000，返回8，5为101，返回1。6为110，返回10（2），即2.
{
    return i&(-i);
}
int change(int i,int k)//修改函数，即将第i个数据增加k以后用于更新tree数组和前缀和，也是用来建树的函数
{
    while(i <= n)
    {
        tree[i] += k;
        i += lowbit(i);
    }
}
int search(int i)//查询函数，可以用于查询前i个数的和
{
    int sum = 0;
    while(i)
    {
        sum += tree[i];
        i -= lowbit(i);
    }
    return sum;
}
int main()
{
    memset(all,0,sizeof(all));
    memset(num,0,sizeof(num));
    memset(all,0,sizeof(all));
    scanf("%d",&n);
    for(int i =1;i<=n;i++)
    {
        scanf("%d",&num[i]);
        change(i,num[i]);
        all[i] = all[i-1] + num[i];//all数组是用O(n)的复杂度进行求前缀和，作为对照。
    }
    //change(5,5-num[5]);这些语句可以修改tree数组达到修改前缀和的目的，不过num数组和all数组并未更新
    while(1)
    {
        int x;
        scanf("%d",&x);
        printf("%d  %d\n",search(x),all[x]);//进行对照
    }
}
